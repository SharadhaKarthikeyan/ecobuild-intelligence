# EcoBuild Intelligence Dashboard Visuals

This directory contains placeholders for dashboard screenshots when publishing the project to GitHub.

## Suggested Screenshots to Include:
1.  **Portfolio Overview**: Main dashboard page displaying KPI cards and energy/carbon distributions.
2.  **Building Benchmarking**: Interactive comparison scatter plot showing floor area vs energy colored by risk category.
3.  **Risk Profile Drilldown**: High/Critical risk asset tables with dynamic color scale formatting.
4.  **Retrofit Financials**: Payback distributions and prioritized capital improvement lists.
5.  **Individual Profile**: Dynamic monthly expense charts and SQLite equipment listings for single buildings.
6.  **AI Assistant Chat**: Natural language assistant interface answering building performance questions.

## Capture Instructions:
*   Launch the Streamlit application:
    ```bash
    streamlit run app.py
    ```
*   Set browser width to standard desktop layout.
*   Capture screenshots in dark/light mode and save them as PNGs in this directory.
*   Update the main `README.md` to link these images.
